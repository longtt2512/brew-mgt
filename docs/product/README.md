# Product Docs

The current product contract for `brew-mgt`, derived from intent. These are the
living artifacts; ongoing work updates these rather than a monolithic spec.

- [`PRODUCT.md`](./PRODUCT.md) — purpose, capabilities, product rules, surface.
- [`api-contract.md`](./api-contract.md) — intended REST contract (not locked).
- [`domain-model.md`](./domain-model.md) — core concepts / read models.

Changes to the public surface or product rules are `high-risk` and require a
decision record under `docs/decisions/`.
