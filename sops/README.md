# SOP Library

Standard operating procedures — step-by-step recipes that turn the harness's
principles into repeatable actions. Backend-adapted from the OpenAI Advanced
Pack (the Chrome DevTools / frontend SOP is omitted; an API smoke loop replaces
it).

- [`layered-domain-architecture.md`](./layered-domain-architecture.md) — how to
  add a feature without breaking the layer boundaries.
- [`encode-unseen-knowledge.md`](./encode-unseen-knowledge.md) — how to capture
  something you learned so the next agent inherits it.
- [`local-observability-loop.md`](./local-observability-loop.md) — run, observe,
  diagnose, fix loop for backend work.
